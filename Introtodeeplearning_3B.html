Fronteras y limitaciones¶

Las redes neuronales, de forma ms general, se pueden ver como aproximadores universales de funciones.

Teorema de aproximación universal¶

Una red neuronal feedforward con una sola capa es suficiente para aproximar cualquier funcin continua

Este teorema no nos dice nada acerca del nmero de unidades que debemos tener para poder aproximarnos a la funcin ni tampoco cmo es que se determinan los pesos para ello. Las redes neuronales ms modernas son un ejemplo de esto; en 2017 se public un artculo donde se entren a una red neuronal con imgenes cuyas etiquetas eran aleatorias, lo que se encontr fue que en todos los casos, el error de entrenamiento era 0, mientras que el error en los datos de prueba aumentaba a medida que se elegan etiquetas ms aleatorias.

Limitaciones¶

Una arquitectura de red neuronal no es una mquina que puede dar resultados buenos con cualquier dato que se le d. Los resultados dependen mucho del conjunto de datos de entrenamiento, por lo que es ncesario tener buenos datos para poder generalizar despus de una forma adecuada, ya que la red neuronal dar predicciones de acuerdo a los tipos de datos presentados en el entrenamiento, adems de la cantidad de cada tipo de dato

Incertidumbre¶

Tenemos dos tipos de incertidumbres en los modelos de Deep Learning:

  • Incertidumbre aleatoria: Surge por ruido en la imagen, por ejemplo, si a un clasificador le presentamos dos de los objetos en uno, tendr que predecir cul de los dos es la imagen presentada.
  • Incertidumbre epistmica: Surge cuando le presentamos al modelo algn dato que est fuera de nuestra distribucin, pero el modelo sigue dando una salida que est dentro de la distribucin, generando datos errneos.

Ejemplos adversariales¶

Consiste en presentarle a nuestro modelo un ejemplo, el cual clasifica correctamente y aadir algn tipo de ruido, de forma que el resultado es muy similar al ejemplo original, pero debido al ruido agregado, el modelo se equivoca. Esto viene desde el algoritmo de gradiente descendiente, ya que puede que algn peso de alguna capa intermedia tenga mucha importancia.

Algunas limitaciones de las redes neuronales son:

  • Requieren una gran cantidad de datos.
  • Computacionalmente costosas de entrenar y desplegar.
  • Fcilmente se equivocan con ejemplos adversariales.
  • Sujetas a prejuicios de los datos.
  • Complicaciones para codificar los datos.
  • Complejas de optimizar.
  • Usualmente requieren experiencia para disearlas.

Codificación: Grafos¶

Actualmente se est codificando los datos en grafos. La idea es similar a las de redes convolucionales, donde tenemos una red que extrae caractersticas de la entrada y posteriormente va a una red feedforward. Las redes que realizan esto son Graph Convolutional Networks (GCN), en estas redes se representan los datos como nodos conectados. En este caso se usa un kernel de caractersticas que pasa por los nodos del grafo.

Estas redes neuronales pueden extenderse para representar nubes de puntos, o elementos tridimensionales ya que los grafos preservan las relaciones que hay entre nodos.

ML automatizado¶

Trata de automatizar el diseo de redes neuronales. En este tipo de arquitecturas tenemos una red neuronal denominada controlador y una red hija. El controlador propone una arquitectura de red, la cual es entrenada y se obtiene el desempeo para una tarea en especfico; luego el resultado se retroalimenta al controlador y se optimizan las propuestas de red del controlador.
